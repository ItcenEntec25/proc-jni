<hr>

`wsl.exe --update` 는 Windows Subsystem for Linux(WSL) 을 최신 버전으로 업데이트하는 명령입니다.

```
wsl.exe --update
```

<hr>

❌ 결론부터

VirtualBox 안의 Windows VM에서는 Docker Desktop을 정상적으로 사용할 수 없습니다.

이유는 중첩 가상화 + Hyper-V/WSL2 충돌 때문입니다.
